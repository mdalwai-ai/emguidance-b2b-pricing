# EMGuidance Training and Compliance Pricing

A client-facing pricing calculator for the EMGuidance B2B Academy product. Helps prospective customers understand costs across three packages based on their team size.

**Live site:** [https://mdalwai-ai.github.io/emguidance-b2b-pricing/](https://mdalwai-ai.github.io/emguidance-b2b-pricing/)

## Packages

| Package | Description |
|---------|-------------|
| **Essential** | Core access to EMGuidance Academy content |
| **Essential Pro** | Enhanced access with additional features (Best Value) |
| **Enterprise** | Full-featured package for large organisations (minimum 100 users) |

## Features

- Volume-based tiered pricing across 5 user brackets (1–49, 50–99, 100–499, 500–999, 1000+)
- Dynamic per-user and total cost calculation (monthly and annual)
- Feature comparison table across all three packages
- Enterprise minimum user validation (100 users)
- Automatic "contact sales" prompt when annual cost exceeds R500,000
- Fully responsive design — cards stack on mobile, table scrolls horizontally

## Tech Stack

- Single-file HTML/CSS/JS (no frameworks or build tools)
- Styled to EMGuidance Corporate Identity guidelines
- Google Fonts (Open Sans)
- CSS Grid layout with custom properties
- Hosted via GitHub Pages
